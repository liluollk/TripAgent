# 南京本地人视角

## 中山陵怎么玩

中山陵建议早上7点半之前到，景区免费但要预约，周一关闭祭堂和墓室。从博爱牌坊走到祭堂一共392级台阶，慢慢走大概20分钟。中途在"天下为公"陵门那里歇脚，回头能看整个紫金山。如果想避开人流，可以走旁边的音乐台路，那里有孙中山纪念馆，人少清静。音乐台门票10元，鸽子群飞起来特别好看，适合拍视频。

拍照最佳位置是祭堂前的平台，能拍到对称的蓝瓦建筑和远处山景。下午4点后光线柔和，人也会少很多。中山陵和灵谷寺、明孝陵是联票，可以一天玩完。

中山陵附近吃饭推荐灵谷素面（灵谷寺内），28元一碗，汤头鲜甜。或者去下马坊地铁站旁的"南京大牌档"，人均60元，能吃到美龄粥、天王烤鸭包，本地人也常去。

## 明孝陵怎么玩

明孝陵秋天最美，11月中下旬石象路变成"最美600米"，银杏和红枫交织。早上6点半就开门，晨雾的时候像仙境。不要去太晚，不然全是旅游团的丝巾大妈。明孝陵的核心是神道，分成石象路和翁仲路，全程1.5公里。不要只走主路，旁边的梅花山春天开满梅花，孙权纪念馆也藏在里面。

明孝陵最出片的地方是石象路的骆驼和大象，从东往西拍能带出晨光。还有四方城（洪武大帝神功圣德碑楼），顶没了但很有废墟感。陵宫里的明楼可以登上去，能看到整个钟山风景区。

明孝陵附近吃饭可以去"中山陵茶厂"旁边的小吃摊，卖梅花糕和雨花茶，价格公道。想吃正餐就去苜蓿园地铁站的"刘长兴"，百年老店，小笼包和馄饨都只要十几块。

## 玄武湖怎么玩

玄武湖本地人叫它"后湖"，不要只走解放门到台城那段游客线。从情侣园进，沿着十里长堤走到太阳宫，人少还能看紫峰大厦倒影。环湖路一圈9公里，可以租自行车或坐游览车。最推荐的是翠洲和梁洲，那里有古老的盆景园和湖神庙，游客极少。

看日落去台城上的城墙（门票15元），能看到玄武湖、紫峰和紫金山同框。夜游玄武湖也很有意思，灯光亮起来后，在菱洲坐鸭子船，30元半小时。

玄武湖附近吃饭去湖南路狮子桥，虽然不如以前热闹，但"大牌档"和"狮子楼"还在，盐水鸭和鸭血粉丝汤正宗。或者去科巷菜场，那里有"鸿福面馆"的皮肚面，分量超大，20元够两人吃。

## 总统府怎么玩

总统府一定要请讲解或者租讲解器，否则就是看房子。建议从东苑进入，先看行政院、陶林二公祠，再中轴线看礼堂、八字厅、子超楼，最后西花园看孙中山临时大总统办公室。子超楼的二楼蒋中正办公室要仔细看，墙上地图还保留原样。

总统府附近的大行宫广场有"科巷菜场"，那里藏着无数老字号："草鸡蛋现做蛋糕"的糯米蛋糕、"福印板栗"的糖炒栗子、"张果佬"的煎饼。中午可以去吃"南京精菜馆"，人均80元，推荐炖生敲（鳝鱼）和狮子头。

## 南京吃什么（本地人私藏）

烤鸭不要去夫子庙的网红店，本地人买"徐家鸭子店"（秦虹南路）或"章云板鸭"（评事街）。烤鸭要带卤子吃，皮脆肉嫩。盐水鸭选"韩复兴"（湖北路），老字号，肉质紧实不咸。鸭血粉丝汤不要去回味这种连锁，去"金原鸭血粉丝汤"（锁金村），汤头浓有鸭油，粉丝是山芋粉，特别有嚼劲。

皮肚面吃"易记面馆"（明瓦廊）或"项记面馆"（陆家巷），皮肚发得厚，吸满辣油。一定要加油渣和溏心蛋。锅贴吃"李记清真馆"（评事街），牛肉锅贴外皮金黄酥脆，咬开有汤汁，配一碗牛杂汤，绝配。

糖芋苗和赤豆元宵去"芳婆糕团店"（王府大街），早上六点就排队，乌饭团加老油条也是招牌。梅花糕去"左师傅梅花糕"（马台街），豆沙馅烫嘴但好吃。蒸儿糕在科巷菜场门口有老奶奶推车卖，糯米夹芝麻糖，快要失传了。

小吃集中地推荐：科巷菜场（性价比最高）、茶南农贸市场（地道老城南）、能仁里菜场（电影《我不是药神》取景地，烟火气极浓）。不推荐夫子庙、老门东主街的店，全是游客价。

## 南京怎么出行

禄口机场到市区坐S1号线到南京南站转1/3号线，全程50分钟7元。打车到新街口约120元（含高速费）。南京南站是高铁站，地铁1/3/S1/S3线交汇，别走错出口。

市内交通用地铁，支付宝领"南京地铁电子卡"直接刷。早晚高峰避开1号线（鼓楼到南京站段）、2号线（新街口到苜蓿园段），挤不上车。共享单车很多，但玄武湖周边的坡多不太好骑。坐公交用"金陵通"乘车码，前门上车后门下车。

出租车起步价11元，高峰期很难打。用滴滴或高德，拼车划算。南京人开车比较"猛"，过马路要小心，绿灯也要看车。

## 什么时候去最好

3月中旬到4月初是樱花季，鸡鸣寺路全是人，建议早上6点半前到。秋天11月中旬到12月初最美，明孝陵石象路、栖霞山红叶都在这个时候。夏天（7-8月）太湿热，体感温度40度，不建议来。冬天（12-2月）湿冷，但几乎没有游客，汤山温泉很舒服。

工作日早上中山陵、明孝陵人最少。周末玄武湖、夫子庙本地人也多。所有景点周一的室内部分基本闭馆，提前查。

## 避坑指南

夫子庙核心区只逛不买，特别是那些"手工"雨花石，都是批发染色的。买雨花石去清凉山公园门口的"雨花石市场"。夜游秦淮河不用坐画舫，80元不值，自己沿贡院街走到东水关遗址，风景一样还没人。

不要相信出租车司机推荐的"南京一日游"，都是拉去购物点的。中山陵门口拉你去"免费"看貔貅的，进去就要花钱。不要买路边挑担子的乌梅（打过蜡），不要买"活珠子"如果不是当地朋友带着。

老门东主街全是连锁店，但它的支巷（如三条营、边营）有真正的文艺小店和咖啡馆。总统府门口卖"总统府邮册"的，不要信，去里面官方邮局买。

去栖霞山看红叶，尽量别周末去，堵车两小时下山。去牛首山，建议买景区车票，走路上下非常累。大报恩寺的琉璃塔可以登顶，但电梯要单独买票，不推荐，下面看更好看。
